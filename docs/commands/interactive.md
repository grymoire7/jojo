# jojo interactive

Launch the TUI (terminal user interface) dashboard for managing applications.

## Usage

```bash
./bin/jojo interactive
./bin/jojo  # interactive is the default command
```

## Description

The interactive command provides a keyboard-driven dashboard for browsing and managing your job applications. It is the default command when you run `jojo` without arguments.

## Keyboard navigation

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate between applications |
| `Enter` | Select application |
| `q` | Quit |
